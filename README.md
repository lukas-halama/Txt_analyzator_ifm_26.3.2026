# Text Analyzer Project

A robust Python-based command-line tool designed to perform comprehensive statistical analysis on various text inputs. This project was developed with a focus on clean code principles and efficient data processing.

## 🚀 Features

- **Secure Access Control**: Integrated login system to ensure only registered users can access analysis features.
- **Dynamic Text Selection**: Supports multiple text inputs, allowing users to choose specific datasets for processing.
- **Comprehensive Metrics**:
  - Total word count.
  - Case-specific analysis (Titlecase, Uppercase, Lowercase).
  - Numeric string detection and summation.
- **Visual Analytics**: Automatic generation of a frequency distribution bar chart representing word lengths.

## 🛠️ Requirements & Standards

This project adheres to professional Python coding standards as outlined in the project documentation:
- **PEP 8 Compliance**: Maximum line length of 79 characters for readability.
- **Clean Logic**: Optimized conditional statements (e.g., avoiding `== True`).
- **Input Validation**: Robust handling of non-numeric or out-of-range user inputs.
